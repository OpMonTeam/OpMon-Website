◆ frameByFrame

bool OpMon::GameLoop::frameByFrame = false
private

If true, the game executes itself frame by frame.

This mode is activated my keeping the F2 key pressed. On release, the mode is disabled. One frame can be passed by pressing the key registered in fbfType.
