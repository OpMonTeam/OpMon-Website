Enumeration Type Documentation

◆ EventTrigger

enum OpMon::Elements::Events::EventTrigger
strong

Used to know when the event must be triggered, relatively from the player.

Enumerator
PRESS 

The event is triggered when the player presses the action key.

GO_IN 

The event is triggered when the player is moving to the tile.

ZONE 

The event is triggered as soon as the player sees it.

BE_IN 

The event is triggered as long as the player is on its position.
