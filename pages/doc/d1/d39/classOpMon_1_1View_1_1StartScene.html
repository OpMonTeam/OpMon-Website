◆ part

int OpMon::View::StartScene::part = 0
private

Indicates the part the introduction scene is currently playing.

  • The first part (part 0) is a dialog introducing the professor.
  • The second part (part 1) is the name input screen.
  • The third part (part 2) is an another dialog with the professor. Any other value than 0,1 or 2 for this variable is invalid.
