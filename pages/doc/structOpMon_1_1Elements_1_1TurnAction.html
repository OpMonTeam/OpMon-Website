Detailed Description

Contains data needed to show an action in a turn in a View::Battle.

TurnAction is like a command sent to View::Battle : according to what's inside, View::Battle will act differently. TurnAction::type is like the command itself, and the other variables the arguments. You do not have to use every variable, some are useless according to the type of action you choose.
