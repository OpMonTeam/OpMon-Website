enum OpMon::Elements::EventTrigger
strong

Defines the multiples way for an event to be triggered.

Enumerator
PRESS 

The event is triggered when the player presses the action key.

GO_IN 

The event is triggered when the player is moving towards the tile.

ZONE 

The event is triggered as soon as the player sees it.

BE_IN 

The event is triggered as long as the player is on its position.
