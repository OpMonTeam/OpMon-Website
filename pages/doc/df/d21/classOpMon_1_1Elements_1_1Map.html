Detailed Description

Defines a specific place in a game, containing the event, the animated objects and the map layers.

To lower the loading time, a map object can be created without initialisation. The data of the map is stored in Map::jsonData, and the map can then be loaded by calling Map::loadMap. Using an unloaded map will crash the game.
