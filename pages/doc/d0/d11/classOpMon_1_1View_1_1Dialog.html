◆ pass()

void OpMon::View::Dialog::pass ( )

Move forward in a dialog, in response to an event like a space key pressed.

If the current dialog lines aren't fully displayed (text animation ongoing), terminate the animation, wait 50ms, then ask for a dialog change. If the animation is over, and there still remaining lines to display, play a sound (dialogPass) then pass to the next lines. If there is no more line to display, the dialog is over.
