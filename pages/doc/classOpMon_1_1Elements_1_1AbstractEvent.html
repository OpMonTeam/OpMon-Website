Detailed Description

The base class of all events.

An event has two major methods :

  • update, which is called each frame.
  • action, which is called when the event is triggered.

The way the event is triggered can be defined with the enumeration EventTrigger. The events directly inherited from AbstractEvent are simple events : they must not depend on any other event. To create events that use other events, see AbstractMetaEvent.
