Detailed Description

Interface of all first-level controllers.

A game screen is handled by the GameLoop. When running, two methods are called: processEvent() is called for each sf::Event detected, then update() is called once per frame.

In addition, suspend() and resume() are called when respectively the controller loose the focus (another controller is added on top) and regain the focus.
