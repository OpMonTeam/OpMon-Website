enum OpMon::View::FormulaMode : unsigned int
strong

The different types of formulas used in MovementData.

Enumerator
POLYNOMIAL 

A polynomial formula. Each element of the array will be interpreted as the coefficient of the variable put to the power of the index of the element in the array. Exemple : {2, 8, 1, 6, 9} will be interpreted as 2 + 8t + t^2 + 6t^3 + 9t^4.

MULTIFUNCTIONS 

A formula using different base fonctions. The interpretation will change according to the chosen function. Exemple : {F_SINUS, 2, 3, -5, F_ABS, -3, 4, -2} will be interpreted as 2sin(3t - 5) - 3·|4t - 2| List of currently available functions : F_NOTHING F_SINUS F_COSINUS F_ABS F_POW See the documentation of each one to see how to use it.
